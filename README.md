# Project Hack

A comprehensive web application designed to help Water Resources and Environmental Engineering students plan, research, and execute their academic projects with ease.

## Framework

Vanilla JavaScript with Tailwind CSS (CDN)

## Features

### 30 Globally Relevant Project Topics
- Water Quality Assessment
- Water Resources Management
- Wastewater Treatment Systems
- Hydrology & Hydraulics
- Climate Change & Sustainability
- Environmental Pollution Control

Each topic includes:
- Detailed description and objectives
- Step-by-step methodology
- Key past research references
- Required equipment and tools
- Estimated project duration
- Global relevance context

### Comprehensive Project Guidelines
- Standard project structure (10 sections)
- Writing best practices (Do's and Don'ts)
- Data collection and analysis methods
- Reference standards (WHO, EPA, ISO)

### Research Resources
- Academic databases (Google Scholar, ScienceDirect, Web of Science)
- Data sources (USGS, UN Water, EPA, World Bank)
- Software tools (EPANET, HEC-RAS, SWMM, QGIS)
- Key journals in water and environmental engineering

## Advanced Features

- **Smart Search**: Search across all topics by title, description, or keywords
- **Multi-Filter System**: Filter by category (6 categories) and difficulty level
- **Interactive Cards**: Click any topic to view full details
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile
- **Clean UI**: Professional interface with soft shadows and calm color palette

## Categories

1. **Water Quality** - Contamination assessment, treatment technologies
2. **Water Resources** - Management, conservation, distribution systems
3. **Wastewater Treatment** - Conventional and nature-based solutions
4. **Hydrology & Hydraulics** - Flood modeling, drainage systems
5. **Climate & Sustainability** - Climate impacts, water-energy nexus
6. **Environmental Pollution** - Soil, water, agricultural pollution

## Difficulty Levels

- **Beginner**: 2-4 months, minimal equipment, suitable for undergraduates
- **Intermediate**: 3-6 months, moderate complexity, standard equipment
- **Advanced**: 4-8 months, sophisticated analysis, specialized tools

## Theme

Clean Light theme with white/gray backgrounds, soft shadows, and professional blue accents for a calm and productive user experience.
